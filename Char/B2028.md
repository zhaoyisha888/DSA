# 反向输出一个三位数
## 题目描述
输入一个三位数，反向输出它的每一位数字。

## 注意事项
1. 输入的是一个三位数，可能以 0 开头，如 001。
2. 所以不能当作整数来处理，必须当作字符串处理，才能**保留前导零**。
3. 反向输出就是将字符串反转。
    - 用字符串处理而不是整数；
    - 可以利用 Python 切片 [::-1] 快速反转；C中使用字符串数组反向循环
    - 保证输入输出长度为3，保留前导0。

## 说明
1. char str[4]; 分配4个字节空间：3位数字 + 1个结束符 \0。
2. scanf("%3s", str); 精确控制只读 3 个字符，防止超出边界。
3. 使用循环 for (int i = 2; i >= 0; i--) 反向输出字符。
4. putchar(str[i]); 逐个输出字符，保留前导零。
