# 方程类型建模问题
假设地球上的新生资源按恒定速度增长。照此测算，地球上现有资源加上新生资源可供 x 亿人生活 a 年，或供 y 亿人生活 b 年。
为了能够实现可持续发展，避免资源枯竭，地球最多能够养活多少亿人？
**经典的资源分配与增长建模问题，可以用线性方程组来求解。** 

# 一、题目建模的关键假设

题目说：
地球上现有资源 + 新生资源，能：

- 供 x 亿人用 a 年，或者
- 供 y 亿人用 b 年。
- 新生资源是 每年恒定增长。

# 二、设变量建模：
我们设：
- R = 地球上当前的资源总量（亿人・年）
- G = 每年新生资源增长量（亿人・年）
- N = 地球最多能持续养活的人数（我们要求的目标，单位：亿人）

# 三、根据题目建立方程组

根据题意：
第一种情况：x 亿人使用 a 年，总共需要资源：x * a
所以总资源要满足：
    R+G?a=x?a
第二种情况：y 亿人使用 b 年，总共需要资源：y * b
所以总资源也满足：
    R+G?b=y?b

# 四、用两个方程消元求 R、G
用上面两个式子联立：
    R+Ga=xa (1)
    R+Gb=yb (2)
用 (2) - (1) 消掉 R：

    Gb?Ga=yb?xa
    G(b?a)=yb?xa
    G=(yb?xa)/(b?a)

然后代入任一式子求 R，例如代入 (1)：

    R=xa?Ga

# 五、要求的是最多能养活多少人（即使得资源永不枯竭）
我们要求的是：年需求 ≤ 每年增长量，即：
    N?1≤G
所以：
    N≤G

? 最多能养活的人数 N 为：
    N = (yb - xa) / (b - a)
?
 
 
? 注意事项
- 要确保 b ≠ a，否则除数为 0（题目应不会出这种数据）
- 答案可以是小数，单位是“亿人”
- 此类题目阅读时不要遗漏解题条件
