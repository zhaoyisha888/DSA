## 题面
学校和家距离 s 米，以 v 米每分钟的速度匀速走向学校。路上额外花费 10 分钟进行垃圾分类。学校要求必须在上午 8:00 到达，不迟到的前提下，最晚出门时间。提前的时间不会超过一天。

### 输入格式
一行两个正整数 s,v，分别代表路程和速度。

### 输出格式
输出一个 24 小时制下的时间，代表 yyy 最晚的出发时间。
输出格式为 HH:MM，分别代表该时间的时和分。必须输出两位，不足前面补 0。

## 解题思路
1. 总时间 = 走路时间 + 垃圾分类时间（10分钟）
2. 走路时间 = s / v（单位是分钟） **这里整形除法返回值为int，会损失时间精度，后面使用强制类型转换**
3. 总耗时 = ceil(s / v) + 10 （因为时间不能舍掉，要向上取整）
4. 从 08:00 倒退 总耗时 分钟，就得到最晚出门时间

## 注意
1. 使用 ceil 要包含 <math.h>
2. ceil(s / v) 会因为整型除法失精度，必须转换为浮点数：ceil((double)s / v)
3. 输出一定要用 %02d 格式化小时和分钟，不足2位补0
>这里 02 参数含义：使用 0 填充（默认是空格填充）；宽度为 2，如果数字不足 2 位，就补足


## **为什么一定要用ceil取整**
```c
int t = (double)s / v;
int t = (int)ceil((double)s / v);
```
想要的是**向上取整（ceil）**的结果，只是**截断小数部分（int向零取整）**，这两者结果不一样，尤其当除不尽的时候会出错。
**注意ceil本身返回为double类型**

| 除法结果 | `int t = (double)s / v` | `int t = (int)ceil(...)` |
| ---- | ----------------------- | ------------------- |
| 1.01 | 1                       | 2                   |
| 1.99 | 1                       | 2                   |
| 3.0  | 3                       | 3                   |

## **拓展技巧：整数向上取整除法**
如下使用可以避免使用 ceil()，纯整数运算，不需要引入 <math.h>，也不会有浮点误差。
```c
int t = (s + v - 1) / v;
```
