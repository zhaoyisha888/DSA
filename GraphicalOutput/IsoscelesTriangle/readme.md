# 典型的 中心对称、逐层递增字符数 的图案
## 描述
构造一个底边长5个字符，高3个字符的等腰字符三角形
```
  *  
 *** 
*****
```

## 观察规律 ―― 构造每一行
拆分这3行观察：

| 行号  | 空格数量 | 字符数量 | 总宽度 |
| --- | ---- | ---- | --- |
| 第0行 | 2    | 1    | 5   |
| 第1行 | 1    | 3    | 5   |
| 第2行 | 0    | 5    | 5   |


发现：
- 每行的字符数是奇数：1, 3, 5（规律：2*i + 1）
- 每行前面空格数是递减的：2, 1, 0（规律：height - i - 1）
- 每行总宽度恒定为 5（说明字符和空格构成对称）


## 是不是所有等腰三角形题目都用“奇数”？

几乎是 ―― 只要是对称图形，并且是字符构造，基本都是奇数对称展开，原因如下
对称结构中，中心点占1格，左右对称，所以总宽度是奇数。
“等腰”的含义，其实就是“中心对称”，所以必须在字符数上逐层构造奇数。


## 思维总结：以后怎么做这种题？

一看到题目是“等腰三角形”、“对称输出”时，思考路线如下：

1. 总共有几层？（高度）
2. 每层字符数多少？（通常是 2*i + 1）
3. 每层左边空格多少？（通常是 height - i - 1）
4. 输出顺序：先空格、再字符、然后换行

甚至可以拿笔画出三角形，数一下空格和字符数，这就是“**模拟 + 找规律**”的思维方式。
