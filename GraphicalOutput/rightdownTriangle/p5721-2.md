## 题目要求输出的三角形矩阵如下（以 n=4 为例）：

```text
      01
    0203
  040506
07080910
```

1. ** 数字规律 ：**
     数字从 01 开始，每行依次递增。
     每行的数字数量 = 行号（第 1 行 1 个数字，第 2 行 2 个数字，依此类推）。

2. **图形规律**
我们需要找到空格数量和数字起始位置的数学关系。

**空格数量**
   - 第 row 行（从 0 开始计数）的空格数量为 2 * (n - 1 - row)。
   - 比如 n=4：
        - 第 0 行：2 * (4 - 1 - 0) = 6 空格。
        - 第 1 行：2 * (4 - 1 - 1) = 4 空格。
        - 第 2 行：2 * (4 - 1 - 2) = 2 空格。
        - 第 3 行：2 * (4 - 1 - 3) = 0 空格。

**数字起始位置**
   - 数字从第 n - 1 - row 列开始打印（列号从 0 开始）。
   - 比如 n=4：
        - 第 0 行：从第 3 列开始打印数字。
        - 第 1 行：从第 2 列开始打印数字。
        - 第 2 行：从第 1 列开始打印数字。
        - 第 3 行：从第 0 列开始打印数字。

## **关键步骤**
1.  初始化数字 ：从 1 开始。
2.  外层循环 ：控制行数（row 从 0 到 n-1）。
3.  内层循环 ：控制列数（col 从 0 到 n-1）。
    - 如果 **col >= n - 1 - row**，打印数字并递增。
    - 否则打印两个空格（因为每个数字占 2 位）。
4.  换行 ：每行结束后换行。

## **执行过程（以 n=4 为例）**
1.  第 0 行 ： row=0：
- col 从 0 到 3：
    - col=0：0 >= 3  否 → 打印空格。
    - col=1：1 >= 3  否 → 打印空格。
    - col=2：2 >= 3  否 → 打印空格。
    - col=3：3 >= 3  是 → 打印 01。
输出：      01（6 空格 + 01）。

2.  第 1 行 ：row=1：
- col 从 0 到 3：
    - col=0：0 >= 2  否 → 打印空格。
    - col=1：1 >= 2  否 → 打印空格。
    - col=2：2 >= 2  是 → 打印 02。
    - col=3：3 >= 2  是 → 打印 03。
输出：    0203（4 空格 + 0203）。

3.  第 2 行 ：row=2：
- col 从 0 到 3：
    - col=0：0 >= 1  否 → 打印空格。
    - col=1：1 >= 1  是 → 打印 04。
    - col=2：2 >= 1  是 → 打印 05。
    - col=3：3 >= 1  是 → 打印 06。
输出：  040506（2 空格 + 040506）。

4.  第 3 行 ：row=3：
- col 从 0 到 3：
    - col=0：0 >= 0  是 → 打印 07。
    - col=1：1 >= 0  是 → 打印 08。
    - col=2：2 >= 0  是 → 打印 09。
    - col=3：3 >= 0  是 → 打印 10。
输出：07080910（0 空格 + 07080910）。
