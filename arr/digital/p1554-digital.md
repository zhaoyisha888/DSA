# P1554 梦中的统计

## 题目描述

Bessie 的大脑反应灵敏，仿佛真实地看到了她数过的一个又一个数。她开始注意每一个数码（$0 \ldots 9$）：每一个数码在计数的过程中出现过多少次？
给出两个整数 $M$ 和 $N$，求在序列 $[M, M + 1, M + 2, \ldots, N - 1, N]$ 中每一个数码出现了多少次。

## 输入格式
第 $1$ 行: 两个用空格分开的整数 $M$ 和 $N$。

## 输出格式
第 $1$ 行: 十个用空格分开的整数，分别表示数码 $0 \ldots 9$ 在序列中出现的次数。

## 输入输出样例 #1
### 输入 #1
```
129 137
```
### 输出 #1
```
1 10 2 9 1 1 1 1 0 1
```
## 说明/提示
数据保证，$1 \leq M \leq N \leq 2 \times 10^9$，$N-M \leq 5 \times 10^5$。



## 正确的数字分解?：

使用temp % 10获取最后一位
使用temp /= 10去掉最后一位
直到temp == 0结束

## 拓展知识总结
**编程题目中，sum[digit]++和count++**
1. 本质区别?
   
|  特征  ?| sum[digit]++  | count++ |
|---|---|---|
|操作对象|数组元素（带索引的统计）|单一计数器变量|
|?内存占用?|需要数组存储空间|只需一个int变量|
|统计维度?|多维统计（按类别/数字/字符等）|单维统计（总数/次数）|
|?典型时间复杂度?|O(n*k)（k为分类数）|O(n)|

1. 典型应用场景对比?
- ?sum[digit]++的经典题型?
   1. ?数字频率统计
    ``` c
    // 统计数字0-9在1~n中出现的次数
    int sum[10] = {0};
    while(n > 0){
        sum[n%10]++;
        n /= 10;
    }
    ```
    2. ?离散化数据统计?
    ```c
    // 统计考试成绩各分数段人数
    int score_range[5] = {0}; // 0-59,60-70,71-85,86-99,100
    for(int s : scores){
        if(s < 60) score_range[0]++;
        else if(s <= 70) score_range[1]++;
        // ...
    }
    ```
- count++的经典题型?
  1. 满足条件的元素计数
   ```c
   // 统计满足条件的元素个数
   int count = 0;
   for(int x : arr){
       if(x > 0) count++;
   }
   ```
   2. 递归/循环次数统计
   ```c
   // 统计素数个数
   int count = 0;
   for(int i=2; i<=n; i++){
        if(isPrime(i)) count++;
    }
   ```
   3. 滑动窗口计数
    ```c
    // 统计满足条件的子数组数量
    while(right < n){
        if(condition) count++;
        right++;
    }
    ```

4. 选择依据?
   
- ?用sum[digit]++当?：

>需要统计多个类别的数据
>需要保留完整的分布信息?
>后续需要频率相关的计算?（如排序、概率）

- ?用count++当?：

>只需知道总数或出现次数?
>条件判断简单（是/否两种状态）
>内存受限时优先使用

**总结表?**
|?维度?|sum[digit]++|count++|
|-----------|----------|-----------|
|数据结构|数组/哈希表|单一变量|
|?空间效率?|较低（需存储分类）|较高|
|?信息量?|保留详细分布|仅总量|
|?适用题型?|频率统计、分布分析|总数统计、存在性判断|
