# 枚举类型
1. 什么是“枚举”？
枚举就是“把所有可能的情况一个个试一遍”。
2. 枚举的思路： 
换句话说，程序穷举所有合法选项，再从中找到满足条件的最优解、最大值、最小值或特定答案。
| 特征                   | 说明                              |
| -------------------- | ------------------------------- |
|  **输入规模小**          | 枚举题通常数据范围不大，能“遍历所有可能”而不超时       |
|  **问题要求最值或满足条件的情况** | 例如“找最大值/最小值”、“找某种条件下最先/最后出现的情况” |
|  **没有复杂的状态变化**      | 枚举题往往逻辑清晰，没有动态规划或贪心那种复杂转移       |
|  **暴力可行，优化不是必须的**   | 即使不用高效算法，直接试也能通过所有测试点           |
